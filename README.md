This is my first extension. So first, open a terminal in the folder where the files are present.

Then run this command to make a folder to hold the extension.
```mkdir -p ~/.local/share/gnome-shell/extensions/unpanel@cannot4071-hub```

After that use this command to copy the files.
```cp -r ./* ~/.local/share/gnome-shell/extensions/unpanel@cannot4071-hub/```

After this,

- On X11 (Xorg / XLibre)
Run Alt + F2, type `r` and hit enter.

- On Wayland
... uh.... oh yeah! Log out and Log back in.

Features:

- Panel goes away when apps go in fullscreen!
- Hot corner will work. I've tested it. I've tried to break it. Trust me bro.
- It won't kill fps. (I tested on my GT 710.)

My recommendations:

- Use hot corner. IT'S JUST TOO BUTTERY SMOOTH.
- Use on GNOME 48, 49 or 50. I will add support for more don't worry!
